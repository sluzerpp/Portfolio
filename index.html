Journey

Journey is a mobile quest game using an interactive map, using a system of quests placed on the map, the user can explore various places in his city.

The application also has an admin panel in which you can conveniently add new quests to the map, as well as create routes from them.

Functional

The application has extensive functionality such as registration, authorization, tracking the user's location, building routes to quests, completing quests, and games such as tests and puzzles. There is also an admin panel for managing the application.
