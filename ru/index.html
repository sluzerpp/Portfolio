Codewars

Проект и приложение, в котором можно найти частичную копию CodeWars. Приложение было разработано в рамках курсового проекта в колледже

Функционал

Была реализована основная функциональность CodeWars, включая выполнение кода в браузере. Возможно выполнение кода на таких языках, как C#, JAVA, TypeScript и JavaScript. Каждый отдельный язык реализован как контейнер Docker на серверной стороне приложение

Jobored

Функционал

GraphiQL

Приложение, которое является полной копией IDE GraphiQL для выполнения запросов GraphQL API по Рику и Морти

Приложение также может работать с любым другим GraphQL API

Функционал

Приложение реализует необходимый функционал для работы с GraphQL API, такой как редактор запросов, переменные, заголовки запроса. Вы также можете просматривать документацию по API. Также имеется авторизация с использованием Firebase

К репозиторию

Journey

Journey - это мобильная квест-игра с использованием интерактивной карты. С помощью системы квестов, размещенных на карте, пользователь может исследовать различные места в своем городе

Функционал

Приложение обладает обширным набором функциональности, таким как регистрация, авторизация, отслеживание местоположения, построение маршрутов к квестам, выполнение задание и тому подобное. Также имеется административная панель для управления приложением
